# AI Агент с Мета-Познанием

Данный проект представляет собой интеллектуального агента с мета-познанием, способного к самоанализу, адаптации и обучению. Система включает в себя передовые технологии искусственного интеллекта и машинного обучения.

## Архитектура

Проект состоит из двух основных компонентов:

1. **Бэкенд**: FastAPI-приложение с интеграцией различных LLM
2. **Фронтенд**: React-приложение с визуализацией мета-когнитивных процессов

## Технологии

- Python 3.9+
- FastAPI
- React
- TypeScript
- PostgreSQL
- Redis
- Docker
- Kubernetes (опционально)

## Подготовка к запуску

1. Клонируйте репозиторий
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   cd frontend && npm install
   ```
3. Создайте файл `.env` на основе `.env.example` и укажите необходимые переменные окружения
4. Запустите приложение:
   ```bash
   # Для запуска бэкенда
   uvicorn api.main:app --reload
   
   # Для запуска фронтенда
   cd frontend && npm run dev
   ```

## Безопасность

Проект включает в себя продвинутые механизмы безопасности:
- Валидация и санитизация входных данных

- Шифрование чувствительных данных

- Фильтрация чувствительных данных в логах
