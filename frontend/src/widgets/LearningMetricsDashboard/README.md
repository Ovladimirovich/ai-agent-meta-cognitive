# LearningMetricsDashboard

Компонент LearningMetricsDashboard предназначен для отображения метрик обучения мета-когнитивного AI агента. Он предоставляет визуализацию производительности агента во времени, статистику использования различных паттернов решения задач, тренды адаптации агента к новым условиям и другие ключевые метрики обучения.

## Структура компонента

Согласно архитектуре FSD (Feature-Sliced Design), компонент расположен в директории `frontend/src/widgets/LearningMetricsDashboard` и состоит из следующих частей:

- `LearningMetricsDashboard.tsx` - основной компонент
- `types.ts` - типы данных для метрик
- `api.ts` - API-клиент для получения метрик
- `hooks/useLearningMetrics.ts` - хук для получения метрик
- `ui/` - подкомпоненты для визуализации
  - `PerformanceChart.tsx` - график производительности
  - `PatternStatsChart.tsx` - график статистики паттернов
  - `AdaptationTrendsChart.tsx` - график трендов адаптации

## Типы данных

Компонент использует следующие типы данных:

- `LearningMetrics` - основной тип, содержащий все метрики обучения
- `PerformanceDataPoint` - точка данных о производительности
- `PatternStat` - статистика по паттернам решения задач
- `AdaptationTrend` - тренд адаптации

## Визуализация

Компонент использует библиотеку Recharts для создания графиков:

- LineChart для отображения динамики производительности
- BarChart для отображения статистики паттернов
- AreaChart/ComposedChart для отображения трендов адаптации

## Интерактивные элементы

Компонент включает в себя интерактивные элементы:

- Выбор временного диапазона (1 день, 7 дней, 30 дней, 90 дней, все время)
- Фильтрация по типам задач (все типы, аналитические, творческие, логические, исследовательские)
- Кнопка обновления данных

## API

Компонент взаимодействует с бэкендом через API-клиент:

- `getLearningMetrics(timeframe, taskType)` - получение метрик с возможностью фильтрации по временному диапазону и типу задач

## Использование

Компонент может быть использован как в составе главной страницы агента (AgentDashboard), так и самостоятельно:

```tsx
import LearningMetricsDashboard from '@/widgets/LearningMetricsDashboard';

// В составе другой страницы
<LearningMetricsDashboard />

// С начальными параметрами
<LearningMetricsDashboard initialTimeframe="30d" initialTaskType="analytical" />
```

## Тестирование

Компонент покрыт unit-тестами, проверяющими:

- Отображение основных метрик
- Функциональность интерактивных элементов
- Правильную работу с разными типами данных
- Корректность отображения графиков

## Интеграция

Компонент интегрирован в главную страницу агента (AgentDashboard) и отображается в правой колонке интерфейса.
